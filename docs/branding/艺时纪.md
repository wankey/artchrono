> ArtChrono · 记录每一刻艺术时光

## 一、名字释义

| 元素 | 含义 |
|------|------|
| 艺 | 艺术培训——领域锚定 |
| 时 | 时间——课时、考勤时刻、续费周期、成长的刻度 |
| 纪 | 记录 · 足迹 · 编年——考勤、缴费、成长轨迹 |

**潜台词**："时纪"音近"时计"（钟表），暗指精准的时间管理——排课、倒计时课时余额、考勤打卡。

---

## 二、英文名

**ArtChrono** — Art × Chronology

- *Chrono* 来自希腊语 χρόνος（时间），常见于 chronology（编年史）、chronometer（精密时计）
- 组合意为"艺术编年"，精准对应"艺时纪"
- 简短有力，国际发音友好，域名可用

备选：
- **ArtLog** — 更轻量，偏"日志"语感
- **TimedArt** — 直译，略生硬

## 四、视觉概念

### 4.1 核心意象

**"刻度"**——用时间刻度丈量艺术成长。

可以具象化为：
- 钟表刻度线围成画笔或音符轮廓
- 时间轴 + 艺术元素（笔触 / 音符 / 舞姿剪影）
- 三个竖线（艺·时·纪三字）组成节拍器、调色刀或谱号

### 4.2 Logo 方向

#### 方向 A：时间刻度 × 画笔

一个圆形钟面，12 条刻度线。其中一条刻度线延长变形为毛笔/画笔的笔锋飞出圆外——象征"时间中绽放的艺术"。

```
   ○
 ╱ ╲     笔锋飞出 → ／
│ ○ │
 ╲ ╱
```

#### 方向 B：三线谱 × 节拍器

三条横线（代表艺·时·纪三个维度或五线谱），左端对齐，右端长度递增，形似声波纹也似足迹渐远。

```
————————
————————————
————————————————————
```

#### 方向 C：字标为主

"艺时纪"三字使用手写书法字体，笔画末端带一点钟表刻度线的细节，字标即为 Logo。适合轻量化、个人品牌调性。

### 4.3 色彩方案

#### 主色调：**青时** — 蓝绿过渡

| 色阶 | 色值 | 用途 |
|------|------|------|
| 时青-浅 | `#E8F4F0` | 背景、卡片底色 |
| 时青-中 | `#5BB5A2` | 按钮、强调、图标 |
| 时青-深 | `#2D8A7B` | 主文字、深色元素 |
| 时青-墨 | `#1A3C38` | 标题、导航栏 |

**寓意**：青色介于蓝绿之间，既有蓝色的理性（管理、精准），又有绿色的生机（成长、艺术）。"青出于蓝而胜于蓝"——暗含教育中师生传承与超越。

#### 辅助色

| 色名 | 色值 | 用途 |
|------|------|------|
| 暖阳 | `#F4A34D` | 提醒标识、续费按钮、CTA |
| 薄暮 | `#E8D5C4` | 次要背景、分隔 |
| 雪白 | `#FAF8F5` | 全局底色 |
| 墨灰 | `#3D3D3D` | 正文 |

#### 语义色（功能）

| 用途 | 色值 |
|------|------|
| 成功 / 已出勤 | `#5BB5A2`（时青-中） |
| 警告 / 课时不足 | `#F4A34D`（暖阳） |
| 危险 / 缺勤 | `#E07A5F` |
| 取消 / 请假 | `#A8B5B2` |

### 4.4 字体

| 用途 | 中文 | 英文 |
|------|------|------|
| Logo / 标题 | 思源宋体 / 站酷文艺体 | Playfair Display |
| UI 正文 | 苹方 / 思源黑体 | Inter / SF Pro |
| 数字（课时、金额） | 等宽：JetBrains Mono | Tabular Lining |

### 4.5 图形语言

- **圆角**：整体 UI 使用柔和的圆角（8px / 12px），传递温暖
- **线条**：用细刻度线作为装饰元素——列表分隔、空状态插画、Loading 动画
- **插画**：极简线条风格，一人一画架 / 一琴一键的剪影，搭配时间刻度线
- **图标**：来自 Lucide（已在项目中），偏线性、轻盈

## 五、产品文案

### 5.1 一句话介绍

> 艺时纪 —— 独立艺术老师的课程管家。排课、考勤、续费，一纪了然。

### 5.2 价值主张（对老师）

**标题**：把心力还给教学，把管理交给艺时纪。

**正文**：
你用微信记课时、用 Excel 算余额、用脑子记谁快续费了。一个老师管 30 个学生，四件事来回切换，焦虑全在琐碎里。

艺时纪帮你做好三件事：
- **今天该上谁的课？**——打开首页，一目了然。
- **还剩几节课？**——自动扣减，余额不足自动提醒。
- **谁该续费了？**——课时 ≤ 2 时醒目提醒，一键记录收款。

一分钟排课，五秒打卡。剩下的时间，留给教学。

### 5.3 功能亮点

| 功能 | 文案 |
|------|------|
| 今日课程 | 打开即见，今天的每一位学生。 |
| 一键考勤 | 出勤 / 缺勤 / 取消，五秒内完成。 |
| 课时倒数 | 自动扣减剩余课时，余额不足醒目提醒。 |
| 续费记录 | 现金/微信收款，一键入账，自动生成未来排课。 |
| 离线可用 | 没网也能用，联网后自动同步。 |
| 数据导出 | 学生名单、缴费记录，一键导出 CSV。 |

### 5.4 空状态文案（App 内）

| 页面 | 空状态 |
|------|--------|
| 今日无课 | "今天没有排课，去喝杯茶吧 🍵" |
| 暂无学生 | "添加你的第一位学生，开启艺时纪。" |
| 暂无课程 | "创建一门课程，比如钢琴、素描、舞蹈。" |
| 暂无缴费记录 | "记录第一笔续费，让时间开始计数。" |
| 课时余额充足 | "所有学生课时充足，暂无续费提醒 ✨" |

### 5.5 微文案（操作反馈）

| 场景 | 文案 |
|------|------|
| 考勤成功-出勤 | "已出勤，剩余 X 节课。" |
| 考勤成功-缺勤 | "已标记缺勤。" |
| 考勤成功-取消 | "已取消，不扣课时。" |
| 续费记录成功 | "已入账，自动生成未来 X 节课排期。" |
| 同步完成 | "数据已同步。" |
| 离线模式 | "当前离线，数据将在联网后自动同步。" |

### 5.6 关于页

**艺时纪**，记录每一刻艺术时光。

为独立艺术老师而生。告别微信群 + Excel，用最轻量的方式管理你的学生、课程与续费。

从一节琴课到一支画笔再到一个舞姿——每一刻的成长，艺时纪帮你记住。

**诗懿，一位艺术老师，用心打造。**
